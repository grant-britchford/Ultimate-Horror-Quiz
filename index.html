How to Play

Instructions


  1. To play the quiz use the play button in the main menu.
  2. To view the high score page, use the high score button in the main menu.
  3. Each page has a home button which when pressed will take you back to the main menu.
  4. Pages have a replay again button.
  5. To play the quiz. read the question then answer using the choice buttons. All questions are timed.
